# 소개
`WebGL 개발 입문` 책의 예제를 웹에서 바로바로 확인해 보기위해서 azure stroage blob에 올려둔것이다. <br>
예제코드 자체의 경로문제도 약간 있고, CORS 문제가 있어서 로컬에서 바로 바로 확인 안되는 문제가 있어서 약간은 수정했다. <br>
천리길도 한걸음 부터라고 데모확인하고 소스코드 한줄한줄 분석해 봐야겠다. <br>
